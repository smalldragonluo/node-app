## node-app
A Node.js app scaffold

## Software requirements
* Node.js v7.6.0 or higher for ES2015 and async function support.

## command
#### start
* `npm start`

#### dev
* `npm run dev`

#### build assets
* `npm run build`

#### unit test
* `npm run test`
